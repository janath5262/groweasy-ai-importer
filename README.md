          # GrowEasy AI Importer

An AI-powered CSV importer that transforms customer data into the GrowEasy CRM format using OpenRouter AI. The application allows users to upload CSV files, automatically map customer information, preview processed results, and search imported records through a modern web interface.

---

      ## Features

- Upload customer CSV files
- AI-powered customer data mapping
- Automatic field transformation into GrowEasy CRM format
- Search imported customer records
- Import summary dashboard
- Responsive and modern user interface
- Backend deployed on Render
- Frontend deployed on Vercel
